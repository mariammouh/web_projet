# 🎬 Film Review Platform

This project is a **school group project** created as part of our coursework. It provides an interactive platform for users to review and discuss films and TV shows, all under the supervision of an admin.

## 🌟 Features

- Submit and read film and show reviews  
- Comment and engage in discussions  
- Admin panel for content moderation and user management  
- Responsive and user-friendly interface

## 🛠️ Technologies Used

- **Laravel** – for back-end development  
- **Bootstrap** – for responsive design  
- **HTML / CSS / JavaScript** – for front-end structure and behavior  
- **MySQL** – for data storage and management

## 👨‍💻 Teamwork

This platform was developed collaboratively by a student group to practice and demonstrate full-stack web development skills.
